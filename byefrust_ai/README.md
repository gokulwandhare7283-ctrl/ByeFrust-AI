# ByeFrust AI

End the confusion. Make confident choices.

ByeFrust AI is a dynamic, context-aware product recommendation engine that conducts a human-like interview to find the perfect laptop or smartphone for you. It uses a deterministic scoring engine for accurate matching and Google Gemini 1.5 Flash to generate personalized pros and cons.

## Setup Instructions

1. Clone the repository.
2. Navigate to the `byefrust_ai` directory.
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Add your Gemini API key to the `.env` file:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```
5. Run the app:
   ```bash
   streamlit run app.py
   ```
